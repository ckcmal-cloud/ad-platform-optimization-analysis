# 광고 플랫폼 데이터 정제 및 도메인 광고 최적화 분석
> 광고목록 44.5만 건, 광고참여정보 1,685만 건, 광고적립 147만 건을 활용해 광고 도메인 라벨링, 어뷰징 탐지, 도메인별 광고 운영 최적화 전략을 도출한 프로젝트

## 프로젝트 목표:
> 광고 데이터의 품질 문제와 성과 왜곡 요인을 제거하고, 정상 트래픽 기준으로 도메인별 광고 운영 전략을 도출하고자 수행

## 주요 결과

- 광고목록 44.5만 건을 정제하여 43.5만 건 규모의 분석용 마스터 데이터 구축
- 적립 로그 540,743건(36.77%)을 어뷰징으로 판정하여 제거
- 2025년 8월 클릭 급증의 93.09%가 단일 매체(mda_539)에서 발생했으며 CVR은 32.10% → 0.08%로 급락
- 참여형은 공급 비중 97.65% 대비 전환 비중은 5.16%에 불과한 구조 확인
- 실행형(48.81%)·설치형(41.50%)이 전체 전환의 대부분을 차지
- 도메인별 광고유형 성과 구조 차이를 통계적으로 검증(χ² p<0.001, Cramer's V=0.36)
- 금융·라이프·엔터는 실행형, 커머스는 참여형 중심 운영 전략 도출


## 인사이트

- 어뷰징 제거 없이 성과를 분석할 경우 광고 효율이 크게 왜곡될 수 있음
- 광고 공급량과 실제 성과는 일치하지 않으며, 공급 구조보다 전환 기여 구조를 기준으로 운영해야 함
- 도메인별 성과 구조가 달라 단일 운영 기준보다 도메인별 최적화 전략이 필요
- 광고 운영은 전체 공급 확대보다 반응 가능성이 높은 광고유형 중심의 선별 운영이 중요
- 데이터 정제 → 어뷰징 제거 → 성과 분석 → 운영 전략 추천의 통합 분석 체계 구축 가능
